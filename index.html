တပ်ဆင်အသုံးပြုနည်း အဆင့် (၆) ဆင့်

အောက်ပါ လမ်းညွှန်ချက်များနှင့် ပုံများကို တစ်ဆင့်ချင်း တိကျစွာ လိုက်နာဆောင်ရွက်ပါ။ (ပုံများကို နှိပ်၍ အကြီးကြည့်ရှုနိုင်ပါသည်)

1 Step 01 / Download

Application ဒေါင်းလုဒ် ရယူခြင်း

ပထမဦးစွာ သင့်ဖုန်း သို့မဟုတ် ကွန်ပျူတာတွင် VPN ချိတ်ဆက်အသုံးပြုရန်အတွက် V2RayTun application ကို ထည့်သွင်း (Install) ရပါမည်။

အရေးကြီး သတိပေးချက် လုံခြုံရေးနှင့် မြန်နှုန်း အပြည့်အဝ ရရှိစေရန်နှင့် အတုအပ Application များမှ ကင်းဝေးစေရန် တရားဝင် (Official) Application ကိုသာ ဒေါင်းလုဒ်လုပ်၍ အသုံးပြုပါ။
စနစ်အလိုက် ဒေါင်းလုဒ်ရယူရန် ရွေးချယ်ပါ -
Android iOS Windows macOS Linux
ပုံကိုနှိပ်၍ အကြီးကြည့်ရန်
2 Step 02 / Copy Key

VPN Key (ကုဒ်) ကို ကူးယူခြင်း

Admin မှ သင့်အား ပေးပို့ထားသော VLESS key ကုဒ်စာသား (vless://...) အပြည့်အစုံကို Copy (ကူးယူ) လုပ်ပါ။

အထူး သတိပြုရန် ကုဒ်စာသားကို အစမှ အဆုံးထိ စာလုံး သို့မဟုတ် စာကြောင်း တစ်ခုမျှ မကျန်စေဘဲ အပြည့်အစုံ (Completely) ကူးယူရပါမည်။ စာလုံးတစ်လုံးမျှ ကျန်ခဲ့ပါက ချိတ်ဆက်မှု လုံးဝ (လုံးဝ) အောင်မြင်မည် မဟုတ်ပါ။
vless://a46890ef-fe6b-4cff-b4f8...#Vortex Copy အပြည့်အစုံကူးပါ
ပုံကိုနှိပ်၍ အကြီးကြည့်ရန်
3 Step 03 / Add Configuration

"+" Button ကို နှိပ်ခြင်း

ဖုန်း/ကွန်ပျူတာရှိ V2RayTun app ကို ဖွင့်ပါ။ ထို့နောက် မျက်နှာပြင်၏ ညာဘက်အပေါ်ထောင့်ရှိ "+" (Add) button ကို နှိပ်ပါ။

သတိပြုရန် အခြားသော Settings များကို လုံးဝ ပြုပြင်ပြောင်းလဲခြင်း မလုပ်ဘဲ ညာဘက်အပေါ်ထောင့်ဆုံးရှိ "+" button ကိုသာ တိကျစွာ နှိပ်ပါ။
ပုံကိုနှိပ်၍ အကြီးကြည့်ရန်
4 Step 04 / Import Key

Clipboard မှ Import လုပ်ခြင်း

"+" button နှိပ်ပြီးနောက် ပေါ်လာသော မီနူးများထဲမှ ထိပ်ဆုံးရှိ "Import config from Clipboard" ကို ရွေးချယ်နှိပ်ပါ။ သင် Step 2 တွင် Copy ကူးယူထားသော key ကုဒ်မှာ အလိုအလျောက် ဝင်ရောက်သွားပါမည်။

သတိပြုရန် ကုဒ်ကို Manual ရိုက်ထည့်ခြင်း သို့မဟုတ် ပြုပြင်ခြင်း မပြုဘဲ "Import config from Clipboard" ကိုသာ ရွေးချယ်၍ အလိုအလျောက် ထည့်သွင်းပါ။
ပုံကိုနှိပ်၍ အကြီးကြည့်ရန်
5 Step 05 / Connect

VPN ချိတ်ဆက်ခြင်း (Connect)

ကုဒ်စာသား ဝင်ရောက်သွားပြီးနောက် Vortex server ပေါ်လာပါမည်။ မျက်နှာပြင် အလယ်ရှိ ကြီးမားသော Power Button (ခလုတ်အဝိုင်း) ကို နှိပ်၍ ချိတ်ဆက်မှု စတင်ပါ။

သတိပြုရန် (System Permission) ပထမဆုံးအကြိမ် စတင်ချိတ်ဆက်ပါက သင့်ဖုန်း/စနစ်မှ Connection Request (ခွင့်ပြုချက်) တောင်းခံသည့် Box ပေါ်လာပါလိမ့်မည်။ "OK" သို့မဟုတ် "Allow" ကို နှိပ်၍ ခွင့်ပြုပေးပါ။
ပုံကိုနှိပ်၍ အကြီးကြည့်ရန်
6 Step 06 / Success

ချိတ်ဆက်မှု အောင်မြင်ခြင်း

ခလုတ်အရောင်သည် အစိမ်းရောင် (Green) သို့ ပြောင်းလဲသွားပြီး အောက်တွင် Connected > ဟူသော စာသား ပေါ်လာပါက ချိတ်ဆက်မှု အောင်မြင်စွာ ပြီးဆုံးပါပြီ။
